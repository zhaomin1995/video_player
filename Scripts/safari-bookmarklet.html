Awesome Player — Browser Bookmarklet

Drag this button into your browser's bookmarks bar:

Play in Awesome Player

How it works

  1. Drag the button above into your bookmarks bar.
  2. Visit any page with a video (YouTube, Vimeo, an MP4 link, a Plex web page, etc.).
  3. Click the bookmark. The browser fires awesomeplayer://<page-url>, which macOS routes to the app.
  4. The app extracts the page URL, runs it through yt-dlp if it's not a direct media link, and starts playback.

Works in

  • Safari, Chrome, Firefox, Edge — anything with a bookmark bar that runs JavaScript.
  • For Safari, you may need to confirm the first launch (System Settings → Privacy & Security).

If the page hosts the video directly (e.g. an MP4 in a tab)

The bookmarklet prefers the visible <video> element's currentSrc over the page URL, so it'll grab the actual stream URL when there is one — useful for sites that hide the direct URL behind a player iframe.

Alternative: Services menu

Select any video URL anywhere in macOS (Safari address bar, Notes, Mail), right-click, and choose Services → Play in Awesome Player. No bookmarklet needed. After installing the app, run /System/Library/CoreServices/pbs -update in Terminal to refresh the Services registry if the menu item doesn't appear right away.

Source code

javascript:(function(){
  var url = window.location.href;
  var v = document.querySelector('video');
  if (v && v.currentSrc) url = v.currentSrc;
  var src = document.querySelector('source[src]');
  if (src && !v) url = src.src;
  window.location.href = 'awesomeplayer://' + encodeURIComponent(url);
})();
